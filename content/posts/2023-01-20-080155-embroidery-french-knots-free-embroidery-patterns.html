French knots are a classic embroidery technique that can be used to create beautiful and intricate patterns. French knots are created by wrapping the thread around the needle multiple times before pulling it through the fabric. This creates a small, raised knot that can be used to create a variety of patterns. The number of wraps and the size of the knot can be adjusted to create different effects. French knots can be used to create delicate details, bold outlines, and everything in between.
